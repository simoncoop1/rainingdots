# rainingdots

## Intro
This is a simple python script to make dots rain down the terminal screen. It simulates falling from ~2000m and earth gravity  (9.8m/s^2)

## To run
```
python3 pDots.py

```
for an effect
```
python3 pDots.py --effect
```


for a bouncing effect
```
python3 pDots.py --bounce
```

for help

```
python3 pDots.py -h
